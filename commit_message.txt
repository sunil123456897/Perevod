This commit introduces the functionality to automatically generate dictionary and World Bible proposals using the LLM.

- Integrated calls to these new functions within `process_all_chapters_node` in `src/Perevod/agents/nodes.py`.
- Updated `tool_analyze_inconsistencies` in `src/Perevod/agents/tools.py` to use LLM for identifying inconsistencies in translated texts.
- Added retry logic with exponential backoff to `tool_translate_chunk` in `src/Perevod/agents/tools.py` to handle API quota limits.
- Corrected `SyntaxError` in `src/Perevod/agents/tools.py` related to regular expression.
- Adjusted LLM prompts for proposal generation to request 5-10 most important terms/entities.
