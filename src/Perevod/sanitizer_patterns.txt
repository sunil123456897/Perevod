# sanitizer_patterns.txt
# В этом файле хранятся регулярные выражения для очистки текста.
# Каждое правило должно быть на новой строке.
# Строки, начинающиеся с #, игнорируются.

# Удаляет типичные строки о переводе с известных сайтов (ranobelib, rulate, etc.)
# \b - границы слова, i - игнорировать регистр
(?i)\bперевод(?: выполнен| сделан| взят) с сайта.*
(?i)\b(?:translated by|edited by|proofread by)[\s:a-z0-9_-]+
(?i)прочитано на\.? Rulate\.ru.*

# Удаляет URL-адреса
https?://[^\s/$.?#].[^\s]*

# Удаляет стандартные фразы о бесплатном доступе
(?i)глава доступна бесплатно.*
(?i)приятного чтения!

# Удаляет строки, состоящие только из разделителей типа ***, ---, ===
^\s*([*=-])\1{2,}\s*$
